# Uber Trip Analysis Dashboard (Excel)

## Project Overview
This project presents an interactive Excel dashboard built to analyze 103,000+ Uber trip records.
The dashboard helps identify travel patterns, peak hours, high-demand locations, and trip trends
to support data-driven decision-making.

## Key Features
- Interactive Excel dashboard
- Peak hour and time-based analysis
- High-demand location insights
- Date-wise and trip-wise filtering
- Slicers for dynamic data exploration

## Tools & Technologies
- Microsoft Excel
- Pivot Tables
- Pivot Charts
- Slicers
- Data Cleaning and Formatting

## Dataset
- Records: 103,000+ Uber trips
- Fields include:
  - Date and Time
  - Pickup Location
  - Trip Count
  - Day and Hour Analysis

Dataset is used only for educational and analytical purposes.

## Dashboard Insights
- Identified peak travel hours
- Analyzed weekday vs weekend trends
- Found high-demand pickup zones
- Improved visibility into trip distribution patterns

## How to Use
1. Download the Excel file from the repository
2. Open it in Microsoft Excel (2016 or later recommended)
3. Use slicers to filter the data dynamically
4. Explore insights using interactive charts
